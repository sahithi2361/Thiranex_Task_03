# Predictive Analytics Using Historical Data

## 📌 Project Overview

This project focuses on building a predictive analytics model using historical retail data to forecast future pricing trends. The project involves data preprocessing, feature engineering, regression modeling, evaluation, and visualization.

The model uses machine learning techniques to analyze historical patterns and generate accurate predictions for retail pricing behavior.

---

## 🎯 Objective

* Build a predictive model to forecast future trends
* Clean and preprocess historical datasets
* Apply regression-based machine learning algorithms
* Evaluate model performance using standard metrics
* Visualize prediction results and trends

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* XGBoost
* Joblib
* FastAPI
* Ngrok

---

## 📂 Dataset

The dataset contains historical retail information including:

* Product details
* Retail prices
* Competitor pricing
* Discounts
* Seasonal trends
* Time-based sales information

---

## ⚙️ Data Preprocessing

The following preprocessing steps were performed:

* Removed missing/null values
* Converted date columns into datetime format
* Extracted month and year features
* Created engineered features:

  * Average competitor price
  * Price difference
* Encoded categorical variables
* Selected important features for model training

---

## 🤖 Machine Learning Model

### Model Used

* XGBoost Regressor

### Why XGBoost?

XGBoost is a powerful regression algorithm known for:

* High prediction accuracy
* Fast performance
* Ability to handle complex datasets
* Better generalization

---

## 📊 Model Evaluation

The model performance was evaluated using:

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

These metrics help measure prediction accuracy and model reliability.

---

## 📈 Visualization

Visualizations included:

* Actual vs Predicted Values
* Trend Analysis Graphs
* Feature Importance Charts

These graphs help understand model behavior and prediction performance.

---

## 🚀 API Deployment

The trained model was deployed using:

* FastAPI
* Ngrok

This allows real-time prediction through API endpoints.
